# Mobile Pilot Farmer Onboarding

Use this guide for a small internal pilot with 3-5 farmers after the Android preview build passes the release smoke test.

## What To Tell Farmers

This pilot tests whether quick farm notes are useful in real field work.

Farmers can:

- set up basic farm places and items;
- record a quick voice note;
- add optional photos;
- review saved notes later;
- record manual harvest, material-use, and inventory-count records;
- create recovery files they choose where to save or share.

## Important Limits

The app does not:

- transcribe voice notes;
- interpret photos;
- upload data automatically;
- synchronize across devices;
- provide accounts or login;
- back up data to the cloud;
- publish listings or messages;
- restore from a recovery copy yet.

## Suggested First Session

1. Install the internal Android build.
2. Create the farm name.
3. Add a field, bed, greenhouse, or storage area.
4. Add one crop, material, and countable item.
5. Record one voice note with a photo.
6. Review that note in the timeline.
7. Create a media recovery package.

## Plain-Language Privacy Note

Your notes, photos, and farm records are saved on this phone. The app does not send them anywhere automatically. If you create a recovery file, you choose where to save or share it. Keep recovery files somewhere you trust because they contain private farm information.
