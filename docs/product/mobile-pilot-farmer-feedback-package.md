## Pilot Success Criteria

- 3-5 farmers can install and open the internal Android build.
- At least 3 farmers can record a voice/photo farm note with minimal help.
- At least 3 farmers can find the note later in the timeline.
- Farmers understand that notes and photos stay on the device unless exported.
- The owner can create and inspect a media recovery ZIP package before relying on captured media.

## Do Not Add During This Pilot

- AI transcription or photo interpretation.
- Accounts or login.
- Server upload or synchronization.
- Cloud backup or remote restore.
- Telemetry or analytics.
- Shared listings, messaging, or publication.
