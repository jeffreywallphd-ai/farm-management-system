# Infrastructure

This folder is for adapters that connect application ports to concrete local technologies selected for Mobile Pilot 1.

Allowed current directions are SQLite through `expo-sqlite`, versioned JSON export through Expo FileSystem and Expo Sharing, and Zod boundary validation.

Do not add server APIs, sync clients, auth providers, cloud backup, ORM, AI/camera/audio, analytics, or telemetry without later ADR and scope authorization.
