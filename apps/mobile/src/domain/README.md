# Domain

This folder contains small TypeScript domain types for Mobile Pilot 1 concepts.

It may define names, fields, and shared concepts for the three accepted manual records, quantities, farm references, and privacy classification. It must not contain UI, SQLite access, export generation, network calls, or framework-specific code.

Canonical authority remains in `docs/domain/mobile-pilot-1-operational-records.md`, `docs/product/mobile-pilot-1-implementation-scope.md`, and `docs/domain/glossary.md`.

Deferred: AI provenance, shared listing records, server sync state, authentication, and non-Pilot-1 operational records.
