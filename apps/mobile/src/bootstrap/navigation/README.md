# Navigation

This folder is for future app-level navigation composition that does not belong in Expo Router route files.

Do not implement farm workflows here. Workflow behavior belongs behind application use cases, and screen components belong under `src/ui`.

Governed by `docs/product/mobile-pilot-1-implementation-scope.md`, `docs/product/field-workflows.md`, and `docs/standards/accessibility-and-field-usability-standards.md`.

Deferred: server sync screens, AI capture flows, authentication, shared listings, messaging, and deployment-mode UI.
